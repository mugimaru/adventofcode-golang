# [Advent of Code](https://adventofcode.com/): [Go](https://golang.org/)

## Build

```sh
make
```

## Run

Solve first puzzle of year 2019

```sh
./aoc
```

Solve second puzzle of year 2020

```sh
./aoc -year 2020 -day 2
```

Custom input

```sh
./aoc -year 2019 -day 3 -path /path/to/custom/input.txt
```
